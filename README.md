# 🃏 Mnemonica Trainer

Kartengedächtnis-Trainer für den Tamariz Mnemonica Stack mit Major System.

## Features
- Kartentraining: Karte↔Bild, Position↔Karte
- Major System Training: Zahl↔Bild (00–99)
- Sprach-Dialog-Modus (Chrome)
- Filter nach Farbe, Kartentyp, Stack-Bereich
- Statistik mit Trefferquoten
- Kartenbilder & Major System editierbar
